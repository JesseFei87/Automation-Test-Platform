# Prompt Template: ICM Bind Device to Server

Use `@chrome` to execute `TC-ICM-007`.

## Required Inputs

- target_server: `<fill target server name>`
- bind_device: `Test_Ins01`

## Required Steps

1. Open `ICM > 服务器信息`.
2. Click `修改` for the target server.
3. Search for `Test_Ins01` in the bound device selector.
4. Check the device and confirm the save.
5. Keep 3 screenshots: server list entry, selected device in dialog, final saved state.

## Output Format

- case name
- environment
- preconditions
- key steps
- result

- screenshot paths
