# Prompt Template: ICM Device List Query

Use `@chrome` to execute `TC-ICM-003`.

## Inputs

- device_keyword: `<fill query keyword>`

## Required Steps

1. Open `Device Ops > Device List` in a logged-in session.
2. Wait for the list page to render.
3. Enter `device_keyword` and run the query.
4. Decide whether the query action completed successfully.
5. Keep 3 screenshots: list entry, post-query action, final result page.

## Output Format

- case name
- environment
- preconditions
- key steps
- result
- failure point
- screenshot paths
