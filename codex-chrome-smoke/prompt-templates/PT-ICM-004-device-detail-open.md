# Prompt Template: ICM Device Detail Open

Use `@chrome` to execute `TC-ICM-004`.

## Required Steps

1. Start from the device list page.
2. Select the first usable row.
3. Open `Detail` or the equivalent visible entry.
4. Decide whether the detail appears as a page, modal, or drawer.
5. Keep 3 screenshots: list page, after detail click, final detail state.

## Output Format

- case name
- environment
- preconditions
- key steps
- result
- failure point
- screenshot paths
