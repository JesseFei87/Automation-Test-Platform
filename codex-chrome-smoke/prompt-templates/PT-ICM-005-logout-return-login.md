# Prompt Template: ICM Logout Return Login

Use `@chrome` to execute `TC-ICM-005`.

## Required Steps

1. Start from the logged-in homepage after loading finishes.
2. Click the top-right avatar and open the user menu.
3. Click `Logout`.
4. Click `Confirm` in the logout dialog.
5. Decide whether the page returned to login and whether account, password, and login controls are visible again.
6. Keep 3 screenshots: homepage, logout confirm dialog, returned login page.

## Output Format

- case name
- environment
- preconditions
- key steps
- result
- failure point
- screenshot paths
